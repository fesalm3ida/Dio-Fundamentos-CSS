Seletores CSS

Bem-vindo!

No CSS nós possuímos os seletores por ID

  • Seletor por Tipo: Busca elementos por uma tag HTML.
  • Selector por ID(#):< Busca elementos através do atributo id.
  • Selector por Classe(.): Busca elementos atraves do atributo "class".
  • Selector por Universal(*): Seleciona todos os elementos HTML.
  • Selector por Atributo([atrib]): Seleciona elementos que possuem um atributo especifico em nosso documento HTML e conseguimos também buscar atributos com um valor especifico.
